## 参数

| 参数     | 说明                 |
|----------|----------------------|
| 空格     | 向下翻一页           |
| 回车     | 向下翻一行           |
| pagedown | 向下翻一页           |
| pageup   | 向上翻一页           |
| /字符串  | 搜索                 |
| n:       | 重复前一个搜索       |
| N:       | 反向重复前一个搜索   |
| :f       | 立刻显示文件名和行数 |
| b        | 翻到第一页           |
| q        | 离开                 |
