# rm 命令 – 删除命令

## 说明

## 实例

```bash
rm -rf * # 删除当前目录所有文件及目录
ls | grep -v ttt | xargs rm
```

## 参数

| 参数 | 说明                                             |
|------|--------------------------------------------------|
| -i   | 删除前逐一询问确认。                             |
| -f   | 即使原档案属性设为唯读，亦直接删除，无需逐一确认 |
| -r   | 将目录及以下之档案亦逐一删除                     |
