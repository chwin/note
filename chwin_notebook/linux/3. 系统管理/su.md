# su 命令 – 切换登录帐号

## 说明

## 实例

```bash
su linuxcool   # 变更至指定用户身份
su - linuxcool # 完全变更至指定用户身份
```

## 参数

| 参数 | 说明 |
|------|------|
|      |      |

    单个减号（-）	完全身份变更
    -c	执行完指定的指令后，即恢复原来的身份
    -f	适用于 csh 与 tsch，使 shell 不用去读取启动文件
    -l	改变身份时，也同时变更工作目录
    -m	变更身份时，不要变更环境变量
    -s	指定要执行的 shell
    --help	显示帮助信息
    --version	显示版本信息
