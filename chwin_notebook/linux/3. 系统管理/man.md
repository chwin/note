# man 命令 – 帮助命令

## 说明

## 实例

```bash
man 1 passwd # 查看 passwd 命令帮助
man 5 passwd # 查看 /etc/passwd 配置文件帮助
```

## 参数

| 参数 | 说明                                 |
|------|--------------------------------------|
| 1    | 用户命令（可执行命令和 shell 程序）  |
| 2    | 系统调用（从用户空间调用的内核例程） |
| 3    | 库函数（由程序库提供）               |
| 4    | 特殊文件（如设备文件）               |
| 5    | 文件格式（用于许多配置文件和结构）   |
| 6    | 游戏                                 |
| 7    | 协议、文件系统                       |
| 8    | 系统管理和特权命令                   |
| 9    | Linux 内核 API                       |
