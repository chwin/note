# cp 命令 - 复制文件

## 说明

将一个档案拷贝至另一档案，或将数个档案拷贝至另一目录。

## 实例

```bash
cp aa bb cc /root
cp -Rf /dev /test
cp /dev/cdrom /home/sunky/mydisk.iso # 制作光盘镜像
cp /dev/cdrom mycd.iso &    # 后台制作光盘镜像
cp some_file_name{,.bkp}    # 快速创建一个文件的备份（扩展名是。bkp）
```

## 参数

| 参数 | 说明                                           |
|------|------------------------------------------------|
| -f   | 强制覆盖同名文件                               |
| -r   | 递归复制整个目录和子目录                       |
| -a   | 尽可能将档案状态、权限等资料都照原状予以复制。 |
| -p   | 复制时保持文件权限                             |
| -b   | 同名，备分原来的文件                           |
| -l   | 链接文件，而不是复制；见下面的说明             |
| -u   | 更新备份文件                                   |
| -v   | 显示详细信息                                   |
