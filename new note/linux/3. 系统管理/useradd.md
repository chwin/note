## 参数

| 参数 | 说明                                       |
|------|--------------------------------------------|
| -u   | 设置UID                                    |
| -g   | 设置主要组                                 |
| -G   | 设置附属组                                 |
| -c   | 设置用户说明                               |
| -d   | 指定用户家目录                             |
| -s   | 指定用户 shell                             |
| -e   | 帐号失效日期，格式为：YYY-MM-DD            |
| -f   | 指定密码是否失效，0表示失效，1表示永不失效 |
