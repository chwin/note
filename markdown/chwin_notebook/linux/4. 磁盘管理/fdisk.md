# fdisk

    fdisk -l # 查看分区状况
    1. 在 console 上输入 fdisk -l /dev/sda ，观察硬盘之实体使用情形。
    2. 在 console 上输入 fdisk /dev/sda，可进入分割硬盘模式。

1. 输入 m 显示所有命令列示。
2. 输入 p 显示硬盘分割情形。
3. 输入 a 设定硬盘启动区。
4. 输入 n 设定新的硬盘分割区。
4.1. 输入 e 硬盘为[延伸]分割区(extend)。
4.2. 输入 p 硬盘为[主要]分割区(primary)。
5. 输入 t 改变硬盘分割区属性。
6. 输入 d 删除硬盘分割区属性。
7. 输入 q 结束不存入硬盘分割区属性。
8. 输入 w 结束并写入硬盘分割区属性。
